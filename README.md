# radleta — Claude Code Plugin Marketplace

Personal plugin marketplace with 40 plugins for AI-augmented development.

## Setup

```bash
claude plugin marketplace add radleta/claude-plugins
```

## Bundles

Install a bundle to get a curated set of plugins in one command.

| Plugin | Description | Install |
|--------|-------------|---------|
| **dev-workflow** | Complete development pipeline: planning, implementation, verification, and commit workflow | `claude plugin install dev-workflow@radleta` |
| **tech-experts** | All technology expert skills in one install | `claude plugin install tech-experts@radleta` |
